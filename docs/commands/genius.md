# Genius commands

### .genius (`.gen`)

Gets the Genius link for the song you're currently listening to, or the one you're searching for.

!!! note "Examples"
    `.gen`

    `.genius`

    `.g Godzilla Eminem`

    `.genius Mac Miller Good News`
